# 🚀 Java Programming

> 📌 Nơi lưu trữ toàn bộ quá trình học tập và phát triển kỹ năng lập trình Java của tôi.

---

## 📖 Giới thiệu

Repository này tổng hợp các bài tập, ví dụ và dự án nhỏ trong quá trình tôi học Java — từ những kiến thức cơ bản đến nâng cao.

Mục tiêu không chỉ là học cú pháp, mà còn là:

* Hiểu bản chất lập trình
* Áp dụng tư duy giải quyết vấn đề
* Làm quen với cách tổ chức code như một lập trình viên thực thụ

---

## 🧠 Kiến thức đã học

### 🔹 Java cơ bản

* Biến, kiểu dữ liệu, toán tử
* Câu lệnh điều kiện (`if`, `switch`)
* Vòng lặp (`for`, `while`, `do-while`)

### 🔹 Lập trình hướng đối tượng (OOP)

* Class & Object
* Đóng gói (Encapsulation)
* Kế thừa (Inheritance)
* Đa hình (Polymorphism)

### 🔹 Xử lý dữ liệu

* Nhập / xuất dữ liệu (Scanner)
* Làm việc với mảng & ArrayList
* Xử lý chuỗi

### 🔹 File & nâng cao

* Đọc / ghi file
* Xử lý exception
* Tổ chức project Java
